Teacher Setup: Make this fully interactive in Canvas

Why is it not interactive when pasted directly?

Canvas's Rich Content Editor sanitizes code and automatically strips all <script> tags when you paste directly into page text boxes. To keep this quiz interactive, you must use one of these two verified methods:

Method A: Upload as a File (Easiest & Recommended)
  1. Save this code onto your computer as quiz.html.
  2. Go to your Canvas Course > Files and upload the file.
  3. Create an Assignment or Page, click Links > Course Files, and select your file.
  4. When students click the link, it will launch as a fully operational, interactive window inside Canvas!
Method B: Embed via iFrame
  1. Upload the quiz.html to your course files as described in Method A.
